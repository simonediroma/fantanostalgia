fbref Serie A — Stagioni storiche

Sitemap Web Scraper (JSON)

  • sitemap-all-seasons.json — tutte le stagioni, parte da questa pagina
  • sitemap-match-reports.json — stagione singola (generico, con [SEASON])
  • sitemap-2000-2001.json — stagione 2000-2001
  • sitemap-2010-2011.json — stagione 2010-2011
  • sitemap-fixtures.json — solo fixtures (senza dettaglio partita)

Guida: dall'installazione allo scraping

1. Installa l'estensione

  1. Vai su Chrome Web Store → Web Scraper e clicca Aggiungi a Chrome.
  2. Apri chrome://extensions, trova Web Scraper e abilita "Consenti accesso agli URL dei file" (serve solo se apri questa pagina da locale con file://).

2. Importa la sitemap

  1. Scarica la sitemap dalla lista qui sopra (consigliata: sitemap-all-seasons.json).
  2. Apri DevTools (F12) → tab Web Scraper.
  3. Clicca "Create new sitemap" → "Import sitemap".
  4. Incolla il contenuto del JSON e clicca "Import sitemap".
  5. Se usi sitemap-all-seasons.json: apri la sitemap appena importata → clicca "Edit metadata" → sostituisci SOSTITUISCI_CON_URL_FILE_LOCALE con l'URL di questa pagina (copialo dalla barra indirizzi, es. https://fantanostalgia-xxx.run.app/webscraper/seasons.html) → "Save sitemap".

3. Avvia lo scraping

  1. Nella sitemap importata, seleziona il selettore player_row dalla lista e clicca "Scrape".
  2. Imposta Request interval: 4000 ms e Page load delay: 3000 ms (rispetta il rate limit fbref).
  3. Clicca "Start scraping" — si apre una finestra Chrome automatizzata. Non chiuderla e non minimizzarla.
  4. Attendi il completamento. Stima: ~7 sec per partita × 380 partite ≈ 45 min per stagione (per tutte e 14 le stagioni: ~10 ore).
  5. Quando finisce, clicca "Export data as CSV" e salva il file.

4. Converti il CSV

Il CSV di Web Scraper non è nel formato dell'admin. Converti con lo script:

# Windows PowerShell
$env:PYTHONPATH = "."
python -m backend.scrapers.convert_webscraper `
  --input webscraper_export.csv `
  --season 2005-2006 `
  --output fbref_2005-2006.csv

# Linux / Mac
PYTHONPATH=. python -m backend.scrapers.convert_webscraper \
  --input webscraper_export.csv \
  --season 2005-2006 \
  --output fbref_2005-2006.csv

5. Importa i dati

Carica il CSV convertito tramite l'admin: POST /admin/historic/import.
